Bancha Pro: Remote Filtering

ExtJS/Sencha Touch stores can easily be remotely filtered, just set the remoteFilter config to true and filter,
e.g. with store.filter('published',true);. Bancha transforms this into a Bancha request and sends it to the server,
there it configures the PaginationComponent and filters without any work by the developer.
See the javascript code of this example.

var store = Ext.create('Ext.data.Store', {
    model: Bancha.getModel('Article'),
    remoteFilter: true // activate remote filering
});
store.filter('published', true); // just use any remote filtering
store.load();


var store = Ext.create('Ext.data.Store', {
    model: Bancha.getModel('Article'),
    remoteFilter: true // activate remote filering
});
store.filter('title', 'Titel 01'); // just use any remote filtering
store.load();
