Bancha creates the Stores - ExtJS the Rest

The code: cakephp default controller augmentation, javascript code. Tell us how you like it in the Forum.

var store = Ext.create('Ext.data.Store', {
    model: Bancha.getModel('User'), // yes, that's all you have to do
    autoLoad: true // (proxy, fields. etc. is already configured)
});
var pagingStore = Ext.create('Ext.data.Store', {
    model: Bancha.getModel('Article'),
    autoLoad: true,
    pageSize: 10, // just add a paging size and Bancha does everything
    remoteSort: true // Bancha also supports remote sorting
});
Ext.create('Ext.form.Panel', {
    api: { // tell the form to use Banchas methods
        load: Bancha.getStub('User').read,
        submit: Bancha.getStub('User').submit
    },
    ...
}
